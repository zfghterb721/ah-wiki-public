# Magic: The Gathering WPN

Wizards Event Link is the official tournament software for running Magic the Gathering events. It is run through a website and has a downloadable app called Companion for players to use while you run the event.

# Event Scheduling

To schedule a magic event first you must go to the website listed below and log in using your location's credentials.
[Wizards Event Link](https://eventlink.wizards.com)

Once logged in you should see a screen that looks like the one below. Now click on which location you are trying to schedule an event for.

![](/files/104431617.png)

Once you have clicked on the correct location you should see a window that looks like the one below. To create an event click on the Create New Event button located at the top of the screen.

![](/files/104497153.png)

A popup that looks like the image below should appear on the right of the screen. The templates are great for scheduling events that are Wizard sponsored such as Prerelease. Just click the create button on the event template you wish to use and it will bring up the template with the prefilled information. To create a custom event just click on the Create Custom button.

![](/files/104628225.png)

Once you have either clicked on a template or created a custom you should see the pop-up look like the image below. Now enter in the correct information for the event or any missing information from the template. Once all the information has been added click on the Create Event button the pop-up will disappear and you should be able to see the created event on the calendar.

![](/files/104562689.png)

# Running Tournament

## Opening the Event

To run the event you will first need to login to the website as you would do to create the event. Once you have logged in to the correct location click on the event you are running on the calendar. A pop-up should appear on the right side of the screen that looks like the image below. To open the event click on the Open Event button.

![](/files/104464388.png)

## Adding Players

Now that you have opened the event you should see a window that looks like the image below. The next step is to add the players to the event. Players can either register themselves through the Companion App or be added manually. For a player to register through the app they will need a Wizards account (this is the same as their arena account) and a player may create an account in the app if they do not have one. For players to register themselves through the App they will also need the event code. The code is found on the top right of the screen (for this event the code is XX8NJP) and the code can be displayed by pressing the box symbol found next to the event code which will bring up a new window for you to display the code and see what players are registered. Once a player has input the event code in the app their name should appear in the Companion App Lobby. To fully register them you will have to click on their name. the name should disappear from the Companion Event Lobby and should now be visible in the Registered Players section. Players that do not have the Companion App will be required to be entered manually. If the player has a Wizards account entering the email linked to the account in the Add Email tab will add the player to the event. If the player does not have a wizards account click on the Add Guest tab to enter their first and last name and they will be added to the event.

![](/files/104693761.png)

## Starting Round 1

Now that all the players are registered for the event their names should appear like the image below. (Guest will be in pink text and wizard accounts will be in blue text) If the event requires a top 8 cut check the box before starting the first round (If you already selected this option in event creation the box will already be checked) Once all the players are confirmed we can start round 1 to create the pairing click the Start Event button.

![](/files/104562695.png)

## Software Features

Now that the event has started the window should now look like the image below. All players with the app should now be alerted with their table number and match opponent. To inform the players that do not have the app of their match, you can display all pairing and the round timer. to do so click on the same button in the top right corner you used to display the event code, this should open a new window with the timer on the right and the pairing on the left which will auto-scroll to go through all matches. This window will also display match results as they are entered for players to double-check that everything was reported properly. Once all the players have found their seats you may start the round and timer by pressing the Start Round Timer located at the bottom right of the screen. There may be some other materials and options you will need to see click on the Options button which looks like a gear on the bottom left of the screen.

![](/files/104726529.png)

## Options Menu

A pop-up menu should appear that looks like the image below. The Print Match Slips option will generate a new window that will allow you to print out match slips for REL Competitive. Print Player List will generate a new window that will allow you to print the list of registered players for them to ensure that the player is registered for the event if the display is not working. Print Pairings By Name will generate a new window that will allow you to print the pairings for the event ordered alphabetically if the display is not working. Print Pairings By Table will generate a new window that will allow you to print the pairings for the event ordered by table number if the display is not working. Print Standings By Name will generate a new window that will allow you to print the standings for the event ordered alphabetically if the display is not working. Print Standings By Rank will generate a new window that will allow you to print the standings for the event ordered by the player's current rank if the display is not working. Select Random allows you to choose a player at random for a door prize. Delete Current Round Data will delete the current round allowing you to go to the previous round and fix any errors that may have occurred. End Event will end the event early if you need to for any reason.

![](/files/104660995.png)

## Reporting Match Results

Players can report their match results through the app but for those who do not have the app, you will have to report the results manually. Just click on the match you want to report and the match information should expand to look like the image below. Now just click the appropriate results to record the results. Also, you may drop players by checking the box and add penalties to players here.

![](/files/104759297.png)

## Ending the round

Once all the matches have been reported you can start the next round. Your window should look like the image below and in the bottom right of the screen press the Calculate Round X Standings.

![](/files/104431623.png)

## Starting the next Round

The window will change and look like the image below. Your display will also change to show the player's standings so they can check that everything was inputted properly. If something is incorrect click the Edit Match Results button to go back and fix the results. If everything is correct click the Create Round X button. Continue doing this until all rounds are completed and entered.

![](/files/104792065.png)

## Submitting the event

Now that the event has ended you must end the event by printing off the final standings. Then click the End Event button.

![](/files/104529923.png)
