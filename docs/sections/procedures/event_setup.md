# Event Setup

Follow these steps to prepare the store and systems before an event begins.

Proper setup ensures events run smoothly, start on time, and provide a good experience for all participants.

---

## Before Event Start

- ☐ Confirm event time and format  
- ☐ Review prize support structure  
- ☐ Ensure all required materials are ready  

---

## Register and Entry

- ☐ Ensure POS system is ready for event entry  
- ☐ Confirm entry fee (if applicable)  
- ☐ Be ready to process player registrations  

---

## Play Space Setup

- ☐ Arrange tables and chairs  
- ☐ Ensure enough seating for expected attendance  
- ☐ Clear play area of clutter and trash  
- ☐ Confirm walkways are clear and safe  

---

## Event System Setup

- ☐ Open event software (if applicable)  
- ☐ Create or load event  
- ☐ Verify event format and settings  
- ☐ Prepare for player entry  

---

## Supplies

- ☐ Ensure basic supplies are available (pens, paper, etc.)  
- ☐ Confirm any event-specific materials are ready  

---

## Player Registration

- ☐ Greet players as they arrive  
- ☐ Enter players into the event system  
- ☐ Collect entry fees  
- ☐ Answer basic questions about the event  

---

## Announcements Before Start

- ☐ Confirm event format and rules  
- ☐ Clearly explain prize support  
- ☐ Announce round structure and timing  
- ☐ Answer any final questions  

---

## Final Check

- ☐ Confirm all players are entered  
- ☐ Verify event details are correct  
- ☐ Ensure play space is ready  

---

## Start Event

- ☐ Begin event on time  
- ☐ Generate pairings (if applicable)  
- ☐ Announce first round  

---

## Important Notes

- Do NOT start the event without announcing prize support  
- Do NOT begin until all players are properly entered  
- Do NOT guess event settings—verify them  

If unsure at any point, ask a manager.

---

Proper event setup creates a smooth experience for both players and staff and helps prevent issues later in the event.
