# Event System

This document provides an overview of the system used to manage events at A&H Games.

The event system is used to organize tournaments, track players, and manage match results.

---

## Purpose

The event system is used to:

- Register players  
- Generate pairings  
- Track match results  
- Maintain standings  

It ensures events are run fairly, consistently, and efficiently.

---

## When It Is Used

The event system is used during:

- Tournaments  
- Leagues  
- Organized play events  

It should be used for all structured events.

---

## Core Responsibilities

Employees using the event system must:

- Enter players accurately  
- Ensure event settings are correct  
- Record match results properly  
- Keep the event moving on schedule  

Accuracy is critical.

---

## Player Registration

- Players must be entered correctly before the event begins  
- Entry fees should be collected during registration  
- Verify player information if needed  

Do not start an event with incomplete or incorrect player entries.

---

## Pairings and Rounds

- Generate pairings through the system  
- Announce pairings clearly  
- Start rounds promptly  

Do not manually create pairings unless directed by management.

---

## Results Entry

- Enter match results accurately  
- Confirm results with players when needed  
- Resolve discrepancies before moving to the next round  

Incorrect results affect standings and prizes.

---

## Event Completion

- Ensure all rounds are completed  
- Confirm final standings  
- Close out the event in the system  

---

## Important Notes

- Do NOT bypass the event system for structured events  
- Do NOT guess or assume results  
- Do NOT allow delays caused by incomplete entries or missing results  

If unsure how to proceed, ask a manager.
