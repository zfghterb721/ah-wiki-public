# A&H Team : Yu-Gi-Oh! OTS


Yu-Gi-Oh! OTS is the official tournament system for running Yu-Gi-Oh! events in stores. It uses two different programs: one to schedule and track event results, and one to actually run the events. Currently only Monroe has OTS Status.

Login Credentials

- Go to this site [https://shp.cardgame-network.konami.net/mt/login#/tournament](https://shp.cardgame-network.konami.net/mt/login#/tournament)
- Login with your locations credentials.

# Event Scheduling

To schedule event you will have to login to the card game network using the link and login above. Once logged in you should see a window that looks like the image below. Now you need to click on the drop down for the Tournaments/Events section located on the left of the window.

![](/files/104955914.png)

Once expanded it should look like the image below. Now click on the Manage Tournaments/Events.

![](/files/105807873.png)

A new window should appear that looks like the image below. Now click on the New Tournament button located on the top right of the screen.

![](/files/105873409.png)

This should bring up a window that looks like the image below. Now fill out all of the Required fields with the correct information. Once you have entered all of the information click the Apply button at the bottom of the form. This will send the event to be approved. Once approved you should see the tournament listed in the tournament/events page.

![](/files/105906177.png)

To acquire OTS packs you will need to fill out the form attached to this link [https://yugiohcard.wufoo.com/forms/ots-tournament-pack-request-form/](https://yugiohcard.wufoo.com/forms/ots-tournament-pack-request-form/)

# Running a Tournament

Even though you will schedule events in an online portal you will run the event though the downloaded software Konami Tournament Software (KTS) and then submit the results through the online portal. For a comprehensive guide to using the Tournament software see the link attached below.

[https://yugiohblog.konami.com/ots_portal/KTS/KTS User Guide Version 3.0.0 2014July16.pdf](https://yugiohblog.konami.com/ots_portal/KTS/KTS%20User%20Guide%20Version%203.0.0%202014July16.pdf)
