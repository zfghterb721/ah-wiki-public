# Organized Play Event Types

## Purpose

This document defines the types of organized play events at A&H Games.

Each event type has different expectations for:
- Structure
- Enforcement
- Staffing
- Player experience

Event types must be applied consistently across all locations.

---

## Scope

Applies to:
- All organized play events
- All employees scheduling or running events
- All locations

---

## General Requirements

All event types must:

- Follow all store policies
- Be run using the correct official system (if required)
- Start on time
- Be run consistently across locations

Event type does not override store rules.

---

## Event Type Categories

Organized play events fall into the following categories:

- Casual Events
- Standard Events
- Competitive Events
- Premier Events

---

## Casual Events

### Definition

Low-structure events focused on participation and community.

Examples:
- Open play
- League nights
- Learn-to-play events

---

### Expectations

- Flexible structure
- Lower enforcement intensity (rules still apply)
- Focus on player experience

---

### Requirements

- Policies must still be enforced
- Judge/event lead must still be present
- Disruptive behavior must be addressed immediately

---

### Restrictions

- Do not ignore rule violations
- Do not allow behavior that disrupts others

---

## Standard Events

### Definition

Regular scheduled events with defined structure.

Examples:
- Weekly tournaments
- Draft events
- Local leagues with standings

---

### Expectations

- Structured rounds and pairings
- Consistent enforcement of rules
- Clear start and end

---

### Requirements

- Use official system when applicable
- Record results accurately
- Maintain consistent rulings

---

### Enforcement

- Apply rules consistently
- Address disputes immediately
- Escalate when required

---

## Competitive Events

### Definition

Higher-stakes events with stricter requirements.

Examples:
- Store championships
- Qualifiers
- League Cups / Challenges

---

### Expectations

- Strict rule enforcement
- Higher attention to accuracy and fairness
- Increased judge involvement

---

### Requirements

- Must follow all sanctioned event requirements
- Must meet minimum player counts (if required)
- Must have required judge/staff roles

---

### Enforcement

- No leniency on rule violations
- Immediate rulings required
- Escalate complex situations to manager

---

## Premier Events

### Definition

Publisher-controlled or high-importance events with strict requirements.

Examples:
- Prerelease events
- Official championships
- Special promotional events

---

### Expectations

- Full compliance with publisher requirements
- Strict adherence to event structure and rules
- High level of organization and control

---

### Requirements

- Must follow all official instructions
- Must use correct systems and reporting
- Must follow prize distribution requirements exactly

---

### Enforcement

- No deviations from required structure
- No modification of official rules or prize support
- Immediate escalation for any uncertainty

---

## Event Classification

The event type must be determined before scheduling.

Employees must:

- Identify the correct event type
- Apply the correct standards
- Not change expectations mid-event

---

## Consistency Across Locations

- Event types must be handled the same at all locations
- Do not redefine event types per location
- Do not apply different enforcement standards

---

## Summary

- Events fall into defined categories
- Each category has specific expectations
- Policies apply to all event types
- Enforcement increases with event importance
- Consistency is required across all locations

This document is enforceable. All employees must follow it.
