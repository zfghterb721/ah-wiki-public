# Tracking Prize Support
In order to properly account for inventory when providing prize support please add the items as normal in the POS system, change the price to $0, and use the "Prize Support" customer in BinderPOS. Below is an example for Evolution Games providing booster packs for a YuGiOh tournament. Note the highlighted parts of the POS that need to be updated.

This should be used for any inventory that has to leave the system for prizes or promotional reasons. Please use the cart note system if you feel the need to leave any details as to what or why a product was ran through this way.
