# Flesh and Blood GEM

GEM is the official tournament software for running Flesh and Blood events. It is run through a website with support for players through their online accounts for running tournaments.

# Event Scheduling

In order to Schedule a Flesh and Blood Tournament, go to the software website linked below and log in using your location's credentials.

Once logged in the site should look like the image below.

![GEM Homepage](/files/104005642.png)

## New Tournaments

To create the tournament, click on the *New Tournament* button found on the home page. A new window will open (shown below). Next, fill out all of the appropriate entry fields: Tournament nickname, Tournament type, Format, Start Time, Tournament description, Event URL (This is the ticket that can be found on our website), and Tournament Address. Click the *Create* button and you will have successfully created an event in GEM Software.

![New Tournament Form](/files/103972869.png)

![Tournament Details](/files/103776274.png)

## Editing

Once you have created the event you should be able to see the event on the homepage of the software. If you notice that any information about the event is wrong, use the edit button to change the field. Also, if you would like to practice using the software when creating the event, check *Make this a test tournament* to create an event that will not be counted as a real event.

![Edit Tournament](/files/104235009%201.png)

# Support Material Acquisition

All Support material is sent through Southern Hobby Games. To acquire the materials contact the Sale Representative before the start of each month for armory kits and for special tournaments contact a week before the event will take place.

# Running Tournaments

## Accessing the event

To access the event you must first log in to the GEM software. Once at the homepage there should be a list of events that have already been scheduled simply find the event name that corresponds with the event you are trying to run. Once you have found the event click the start button as seen in the image below. If you cannot find the event simply create the event using the instructions above.

![Event List](/files/104235009%202.png)

## Adding players

Each player playing in the tournament must have a Player ID in order to participate in the tournament, ensure all participants have one before starting the tournament. If a player does not have a Player ID they can go to the Flesh and Blood website to create an account and get a unique Player ID. Now to add the players to the roster, Flesh and Blood makes this quite easy because the players can register themselves by following the link provided and logging into their accounts. The link for this sample event can be seen in the image below as massive-cheetah. You can post this link in the discord for players to easily accessed as well as have it displayed on the TV. Once they have added themselves to the roster names should start appearing in the Pending Players area of the software. To fully add them to the event double click on their name and they will be completely added to the roster.

![Player Registration](/files/104103941.png)

### Manually adding players

Some players may not have internet access on their mobile devices and may be required to get added manually. To do this simply click on the Add Players button as seen on the image above this should bring you to a window that looks like the image below. Now you can search for the player you are trying to add. You can filter your results by country by simply clicking the drop-down bar and selecting the United States of America as seen below. Then type the player's First and Last name in the Players search bar and click on the name that corresponds with their Player ID. Finally, click the add button to add that player to the roster.

![Manual Player Addition](/files/104136711.png)

### Checking roster

Once the players have been added the tournament screen should look something like the image below. Before starting the first round double-check the player roster is correct. Also, some new buttons have appeared we can discuss. First, there are the Player List and Player List.csv buttons which will produce files you can print out or display for players to ensure they are on the roster. Next is the Seat Randomly button which will create a seating assignment for players that are drafting to ensure fairness.

![Player Roster](/files/103972875.png)

### Starting the first round

Now that all the players are on the roster you can start the first round. Under the Registered Players section, there should be a section called Tournament Rounds as seen in the image below. To create the round one pairings click the Pair Round 1 button. This will create the pairings and allow the players to see their opponent and table number on their individual accounts.

![Round Pairing](/files/104267777.png)

### Reporting Results

As the players finish their matches you will have to record the results of the match. Players can report the results themselves through their account and accept the results you can click the Accept All Reports button to record the results as seen in the image below. To manually record the results simply click the drop-down under the results column that corresponds to the correct table and select the results of the match as seen in the image below.

![Result Reporting](/files/104005636.png)

### Dropping players

If a player wishes to drop from the tournament this must be recorded before the next round starts in order to remove them from future rounds. You do this just like recording a match result but you use the Drops column as seen in the image below.

![Player Drops](/files/104103947.png)

### Ending the round

Once the results have been entered the winner of the match should have their name highlighted in green. At small level tournaments, it is a good idea to call out the winners of the round to double-check all information is entered properly. For larger tournaments match slips should be used to ensure proper match recording. Once everything is confirmed press the back to tournament button as seen in the image below to begin moving to the next round.

![End Round](/files/104235015.png)

### Starting the next round

Once you are back to the tournament page you should notice some new buttons as seen in the image below. To create pairings to start the next round simply click on the Pair Round X button to create the next set of pairings for the event.

![Next Round](/files/104071172.png)

### Additional Information

Let's quickly go over these new buttons so you are aware of what they do and you will not need to use all of them for every tournament you will run. First, there is the Cut to Top X buttons. You will use these for large tournaments where there is a specific payout for each placement. Once you have completed the designated number of rounds according to the player size you will click on one of these to finish out the tournament. Next, we have the Enter Results Button which is for accessing the round specific window after you have left it. Next, the Edit Round button allows you to edit information about the round. The Delete Round button will remove that round so you can access a previous round to fix any errors that might have occurred. The Pairings button will open a window for you to print and display the pairings for that round. Result Slips opens a window for you to print out the match slips for that round. Standings and Standings.csv will give you files to print or display the player's rank after the results of the round are added. Results.csv will generate a file of the results of each match that round.

### Ending the Tournament

Once the tournament has finished, click the Submit Tournament button to upload the tournament to Legend Studios so player points can be awarded to the player for playing in the event.
