# Organized Play Checklist

## Purpose

This checklist defines the required actions for running organized play events at A&H Games.

This is a strict checklist. Do not skip steps.

---

## Scope

Applies to:
- All organized play events
- All employees running or assisting with events
- All locations

---

## Before the Event

### Event Setup

- [ ] Event is created in the correct official system
- [ ] Correct location is selected
- [ ] Event type and format are correct
- [ ] Start time is correct
- [ ] Event is visible and accessible in the system

---

### Compliance Check

- [ ] Event meets all sanctioned event requirements (if applicable)
- [ ] Minimum player requirement is verified (if applicable)
- [ ] Required judge/staff roles are confirmed
- [ ] Event is scheduled within required timeframes (if applicable)

---

### Materials and Setup

- [ ] Prize support is prepared
- [ ] Seating and play space are ready
- [ ] Walkways and exits are clear
- [ ] Tables and chairs are safe and stable

---

### Staff Readiness

- [ ] Assigned judge/event lead is present
- [ ] Staff understands event format and structure
- [ ] Staff understands enforcement expectations

---

## Event Start

- [ ] Event starts at the scheduled time
- [ ] Do not delay for late players

---

### Player Registration

- [ ] All players are entered into the system correctly
- [ ] Player accounts/IDs are verified (if required)
- [ ] No placeholder or incorrect entries

---

### Announcements

- [ ] Event format and structure are announced
- [ ] Round timing is announced
- [ ] Prize structure is announced (if applicable)
- [ ] Expectations for behavior are clear

---

## During the Event

### Event Management

- [ ] Pairings are generated correctly each round
- [ ] Rounds start on time
- [ ] Results are recorded accurately
- [ ] Event progresses without unnecessary delay

---

### Late Registration

- [ ] Only allowed if manageable
- [ ] Late players receive losses for missed rounds
- [ ] Event flow is not disrupted

---

### Rulings and Disputes

- [ ] All disputes go through the judge/event lead
- [ ] Rulings are made promptly
- [ ] Rulings are applied consistently

---

### Policy Enforcement

- [ ] Customer conduct is actively enforced
- [ ] Family-friendly standards are maintained
- [ ] Safety issues are addressed immediately

- [ ] Violations are handled immediately
- [ ] Escalate to manager when required

---

### Event Control

- [ ] Players are not allowed to self-regulate disputes
- [ ] Disruptive behavior is addressed immediately
- [ ] Event environment remains controlled and organized

---

## Player Removal (If Necessary)

- [ ] Match is stopped immediately
- [ ] Player is informed they are removed
- [ ] Do not argue or negotiate
- [ ] Manager is involved if required

- [ ] Player is removed from the system
- [ ] No refund is issued

---

## After the Event

### Finalization

- [ ] Final results are entered correctly
- [ ] Event is properly closed in the system
- [ ] Reporting is completed (if required)

---

### Prize Distribution

- [ ] Prizes are distributed correctly
- [ ] Official prize structures are followed (if required)
- [ ] Additional prizes match what was announced

- [ ] All prize support is processed through POS as $0 transactions
- [ ] "Prize Support" customer is used in POS

---

### Cleanup

- [ ] Tables and play areas are cleaned
- [ ] Trash is removed
- [ ] Store is returned to normal condition

---

### Issue Review

- [ ] Any incidents are reported to a manager
- [ ] Any problems during the event are noted
- [ ] Follow-up actions are identified if needed

---

## Do Not

- [ ] Do not run events outside official systems (if required)
- [ ] Do not delay event start times
- [ ] Do not ignore rule or policy violations
- [ ] Do not allow inconsistent rulings
- [ ] Do not distribute prizes without recording them

---

## Summary

- Prepare correctly before the event
- Start on time
- Maintain control during the event
- Enforce policies at all times
- Close and report the event properly

This checklist is required. All steps must be followed.
