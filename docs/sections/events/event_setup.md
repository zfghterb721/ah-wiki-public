# Event Setup

Event setup must be completed fully and correctly before the event begins. Poor setup leads to confusion, delays, and inconsistent player experiences.

Employees are responsible for ensuring that all logistical, system, and physical requirements are in place prior to start time.

No event may begin in a partially prepared state.

---

## Core Rules

- Events must start **on time**
- All setup must be completed **before player check-in begins**
- Entry fees, prize structure, and rules must be **predefined and not modified**
- Prize support must be **physically set aside before the event**
- Event capacity must be **defined and enforced**
- Employees may not **improvise setup decisions**

---

## Event Configuration

Before the event, employees must confirm:

- Event format (constructed, sealed, draft, etc.)
- Entry fee (fixed, no exceptions)
- Prize structure (fixed, no changes)
- Start time (strict)
- Registration method:
  - In-store
  - Pre-registration (if applicable)

Employees may not:

- Change entry fees
- Accept alternative payment methods (including store credit)
- Modify prize structures without manager approval

---

## Registration Rules

### Pre-Registration (If Used)

- Pre-registered players must:
  - Be recorded in the event system
  - Be confirmed upon arrival

- Employees must:
  - Verify attendance of pre-registered players before start time
  - Release unclaimed spots at start time

---

### Walk-In Registration

- Registration must close at **event start time**

- Employees may not:
  - Add players after the event has started
  - Make exceptions for late arrivals

Late players must be turned away or directed to the next available event.

---

## Capacity Management

Event capacity must be defined before registration begins.

### Capacity Constraints May Include:

- Product limits (e.g., prerelease kits)
- Physical seating limits
- Staffing limitations

### Enforcement

- Once capacity is reached:
  - No additional players may be added
  - No exceptions may be made

- Employees must:
  - Clearly communicate when an event is full
  - Prevent overcrowding

---

## Space Management

Events take priority over casual play.

Employees must:

- Ensure sufficient table space for all participants
- Reallocate tables as needed before event start
- Ask non-event players to move or leave if space is required

When asking customers to move:

- Be direct and professional
- Do not apologize for enforcing space requirements
- Offer alternatives only if available

Employees may not:

- Allow casual play to interfere with event execution
- Delay event start due to space issues

---

## Seating Setup

Seating structure depends on the event type.

### Before Pairings

- Seating may be:
  - Open seating
  - Assigned seating (if required by event type)

### Requirements

- Tables must be:
  - Clean
  - Clearly organized
  - Adequately spaced

Employees must:

- Ensure players can easily locate seating
- Prevent overcrowding at individual tables

---

## Materials and Supplies

Before the event begins, employees must ensure availability of:

- Pairing method (software or manual system)
- Timer (visible or announced)
- Writing materials (if required)
- Match slips (if used)
- Event-specific materials (e.g., prerelease kits)

Employees may not begin the event without required materials.

---

## Prize Support

Prize support must be prepared before the event begins.

### Requirements

- All prizes must be:
  - Counted
  - Verified
  - Set aside and not sold

- Prize structure must:
  - Match the advertised distribution
  - Be communicated clearly to players

Employees may not:

- Adjust prize support during or after the event
- Add or remove prizes without manager approval

---

## Pre-Event Checklist

Before starting the event, employees must confirm:

- [ ] Event is created in system
- [ ] Entry fees are confirmed and enforced
- [ ] Registration is complete and closed
- [ ] Capacity has not been exceeded
- [ ] Seating is arranged and sufficient
- [ ] Casual players have been relocated if necessary
- [ ] Materials and supplies are ready
- [ ] Prize support is set aside and verified
- [ ] Players are present and ready

---

## Start of Event

- Events must begin at the scheduled start time
- No additional players may be added after start
- All players must be included in initial pairings

Employees must not delay start time for late players.

---

## Non-Negotiable Rules

- No late entries after event start
- No use of store credit for entry fees
- No modification of prize structure
- No exceeding capacity
- No incomplete setup

Failure to follow setup procedures is a performance issue.
